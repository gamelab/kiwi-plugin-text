Plugins Module

Defined in: src/text-1.0.0.js:10

Text is a GameObject that is used when you are wanting to render text onto the current State.

Text has width/height and a hitbox, but because text is difficult to measure, these may not be 100% accurate. It does not have an "Input" component either, although you may choose to add one. Be aware of these limitations.
